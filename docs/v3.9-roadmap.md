# v3.9.0 Roadmap

## Theme: Community Contributions + Research Depth

v3.8.0 delivered the platform (MCP server, GitHub Action, attestation registry, telemetry).
v3.9.0 focuses on deepening test coverage through community contributions and research integration.

## Planned Features

### New Test Modules
- **#60 Subliminal Bias Propagation (Thought Virus)** - Novel attack vector: injecting biases that persist across agent sessions without triggering safety filters
- **#15 AutoGen-Specific Attack Patterns** - Speaker selection poisoning, nested conversation escape, message source spoofing (contributed by @alexmercer-ai)
- **#51 OATR Test Fixtures** - x402 identity verification fixtures for X4-021 through X4-025 (community PR welcome)

### Infrastructure
- **#17 Content-Quality Liveness Checks** - Beyond activity-based heartbeats, detect normalization of deviance in agent outputs (ties to NoD paper DOI:10.5281/zenodo.19195516)
- **#77 A2A-Format Log Endpoint** - AIUC-1 §E003 structured incident response logging
- **#74 /.well-known/agent Endpoint** - NIST NCCoE ID-001 compliance

### Quality
- BaseHarness refactor - shared base class to eliminate ~350 lines of duplicated code across newer modules
- count_tests.py in CI - prevent test count drift permanently
- Expand self-test coverage for MCP server, telemetry, and attestation registry

## Community Contribution Guide
